오늘의 쉼표 — 개인정보 처리방침

시행일: 2026년 4월 23일
최종 수정일: 2026년 4월 29일

오늘의 쉼표(이하 "본 앱")는 이용자의 개인정보와 프라이버시를 중요하게 생각합니다. 본 방침은 이용자가 본 앱을 사용할 때 어떤 정보가 기기 내부에 저장되는지, 그리고 해당 정보가 외부 서버로 전송되지 않는다는 점을 투명하게 안내하기 위해 작성되었습니다. 본 앱은 적용되는 개인정보 보호 관련 법령을 준수하기 위해 노력합니다.

1. 수집하는 정보

본 앱은 이용자를 식별할 수 있는 개인정보(이름·전화번호·이메일·주소 등)를 직접 수집하지 않습니다. 회원가입·로그인 절차가 없으며, 사용자 식별을 위한 정보를 요구하지 않습니다.

다음 정보는 이용자의 기기 내부에만 저장되며, 외부 서버로 전송되지 않습니다.

이 데이터는 Google이 운영하는 Firebase 인프라에 저장되며, 제3자에게 공유되거나 광고 타겟팅에 이용되지 않습니다. 본 앱은 광고 SDK를 사용하지 않습니다.

이용자는 다음 방법으로 분석 데이터 수집을 제어할 수 있습니다:

  • Android 시스템 설정 → Google → 광고 → "광고 ID 재설정" 또는 "광고 맞춤설정 선택 해제"
  • 앱 삭제 (해당 기기의 익명 데이터 수집 중단)

2. 기기 권한 사용

본 앱은 아래 권한을 사용하며, 각 용도는 다음과 같습니다.

  • 알림 권한 (Notifications): "매일 한 문장씩" 기능 사용 시 로컬 알림 발송. 이용자가 설정에서 OFF로 변경 가능.
  • 네트워크 권한: 현재는 사용하지 않습니다. 향후 업데이트에서 앱 콘텐츠 갱신 목적으로 사용될 수 있습니다.

카메라·사진·마이크·연락처·위치 정보는 수집하지 않으며 접근하지 않습니다.

3. 공유 기능 이용 시

공유 기능(텍스트 복사, 카카오톡·인스타그램 공유 등)을 이용할 때 이용자가 선택한 화면 이미지 또는 메시지 텍스트만 이용자의 선택에 따라 타 앱으로 전달됩니다. 본 앱은 공유 대상이나 공유 내역을 별도로 기록·저장하지 않습니다.

4. 데이터 보관 및 삭제

모든 데이터는 이용자 기기에만 저장되므로, 본 앱에서는 외부에 보관하는 데이터가 없습니다.

이용자는 언제든지 설정 → "데이터 초기화" 메뉴를 통해 기기에 저장된 모든 데이터(카테고리·북마크·알림 설정·BGM 설정·본 메시지 기록·온보딩 기록)를 한 번에 삭제할 수 있습니다. 앱을 삭제하면 해당 데이터도 기기에서 함께 제거됩니다.

5. 제3자 제공 및 위탁

본 앱은 현재 이용자 정보를 제3자에게 제공하지 않으며, 외부에 위탁하지 않습니다.

6. 향후 변경 가능성

서비스 향상 또는 사용성 개선을 위해 아래 기능이 추후 추가될 수 있으며, 추가 시점에는 본 방침이 업데이트되고 앱 내에서 이용자에게 고지됩니다.

  • 광고 (예: Google AdMob): 비(非)식별화된 광고 식별자(ADID 등) 활용 — 현재 미적용

변경 사항이 발생하면 이용자에게 최소 7일 전에 앱 내 공지 또는 업데이트 공지사항을 통해 안내합니다.

7. 아동 개인정보 보호

본 앱은 만 14세 미만 아동을 주 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다. 아동이 본인 정보를 입력했다는 사실을 인지한 경우 지체 없이 삭제 조치합니다.

8. 이용자의 권리

이용자는 다음과 같은 권리를 가집니다.

  • 기기에 저장된 데이터를 언제든지 조회, 수정, 삭제할 권리 (앱 내 설정 또는 앱 삭제로 실현)
  • 본 방침 또는 데이터 처리에 관한 문의를 제기할 권리

9. 보안 조치

모든 데이터는 이용자의 기기 내부에만 저장되며, 기기 운영체제의 표준 보호 기능과 앱 샌드박스 범위 안에서 보호됩니다.
